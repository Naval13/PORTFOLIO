Layout Rules

Apple-Level Spacing — No Compromises

Border radius

12px cards · 8px inputs · 99px pills · 0 on dividers. One radius, everywhere.

Section padding

80px top/bottom on desktop. 48px on mobile. Max content width: 1100px centered.

Negative space rule

When in doubt, add more space. Crowded ≠ valuable. Apple principle: let content breathe.

Motion Design

6 Animations — All Purposeful

✅ Include these

  • Hero pipeline nodes — SVG paths with animated stroke-dashoffset, plays once on load, then static
  • Count-up stats — Numbers animate 0→value when section enters viewport (IntersectionObserver)
  • Section fade-up — Sections reveal translateY(24px)→0 on scroll enter, once only
  • Card hover lift — translateY(-4px) + border-color change, 200ms ease
  • Nav blur on scroll — backdrop-filter activates after 40px scroll
  • CTA hover glow — box-shadow: 0 8px 24px rgba(240,165,0,0.3) on hover

✕ Never include

  • Auto-typing / typewriter effects
  • Cursor-following particle trails
  • Parallax on every section
  • Loading screens or spinners
  • Entry pop-ups or overlays
  • Background video loops
  • Infinite scrolling marquees (text)
  • 3D tilt on hover (overdone in AI portfolios)

Why Resend for contact form?

Resend is the cleanest, free-tier email API for Next.js. 100 emails/day free. 1 API key, 1 route file, done. Alternative: Formspree (no backend needed) — swap the form action to https://formspree.io/f/YOUR_ID

Full Deploy Flow — GitHub → Vercel → Live

You'll be live in under 10 minutes

1

Build locally & test

Run npm run dev — check at localhost:3000. Run npm run build — zero build errors required before pushing.

2

Push to GitHub

Create a new repo at github.com named naval-portfolio. Push your code. Keep the repo public (Vercel free tier) or private (Vercel has free private deployment too).

3

Connect Vercel

Go to vercel.com → "New Project" → import your GitHub repo. Vercel auto-detects Next.js. Click Deploy. First deployment takes ~2 minutes.

4

Add environment variables

In Vercel dashboard → Settings → Environment Variables. Add: RESEND_API_KEY = your Resend key. Redeploy.

5

Connect your domain

In Vercel → Settings → Domains. Add navalaggarwal.com. Vercel gives you DNS records to add to your domain registrar. SSL is automatic.

6

Every update = auto-deploy

Every git push to main branch triggers a new Vercel deployment. Pull requests get preview URLs automatically. Zero manual work after setup.
