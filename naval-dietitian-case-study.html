Website Strategy

One page, but with a full modal case study. Here's why.

You asked: separate page, or highlight in the main project grid? The answer is a specific hybrid that gives you the best of both.

Option A — Separate /case-studies page ✕

People have to navigate away. Many won't. You lose the narrative flow of the 1-page scroll. SEO benefit is marginal for freelance goals. Feels like a blog, not a brand.

Option B — Card in Work grid + Modal ✓

Card sits in the filterable project grid. Click → full-screen modal with the complete story, flow diagram, results, tech stack. Keeps 1-page structure. Best conversion.

Option C — Featured Case Study section ✓✓

Add a "Featured Client Work" section above the project grid. This project gets a full-width spotlight card with the flow diagram visible — clients see it before even reaching the project cards.

✦ Recommended: Option C + B combined

  • Above the project grid: A "Featured Work" section — one wide card with a mini version of the flow diagram, the outcome numbers, and a "Read the full case study" button that opens the modal
  • Inside the project grid: Also include a card for it (tagged: AI/ML · ETL · Looker Studio) so it shows up in filtered views
  • In the Services section: Reference it as proof: "Like the referral intelligence system I built for a healthcare practice in Melbourne" next to the AI Integration service tile
  • Result: Visitors encounter this story 3 times, each time with more depth — builds trust incrementally without being repetitive

The Problem

A Melbourne dietitian was drowning in referral emails from GPs and specialists. Each email required manual reading, data extraction, and decision-making about booking priority. She had no visibility into her referral pipeline, no way to spot trends in client needs, and no data to inform her content strategy or identify practice gaps. Bookings were being delayed or missed.

The Solution

  • →Automated ETL pipeline extracting structured data from referral emails
  • →AI (LLM) layer analysing referral text for client expectations + personalised recommendations
  • →Transformation into clean, booking-ready rows in Google Sheets
  • →Live Looker Studio dashboard showing referral volume, conversion funnel, practice gaps, and content themes
